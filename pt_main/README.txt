pt_main

Contains the global configurations for the ParallelText project such as:
    - What apps are currently enabled in the system
    - Where to find the apps relative to the project directory structure
    - Where to find non-app contents (templates, statics, media, etc.)
    - Where to find database file (for SQLite) or how to connect to database (for MySQL, PostgreSQL, etc.)
    - How to map apps to the server's web address

Contains non-media contents:
    - Templates
    - Static files (JavaScript, CSS, images, etc.)
    - Media files (user uploaded contents)
    - Database files (for SQLite)
